D O S

The Dumb OS (DOS) is a Python-based shell with DumbBot AI integration and a text-based GUI mode/file browser. Inspired by the wonky logic of SLMs and the original MS-DOS.

DUI

The Dumb User Interface (DUI) is the Text-Based GUI shell thing inspired by the 2002 iShell for the TI-86.

DumbBot

An SLM (Small Language Model) with a wonky vocabulary. See dumbbot.py for more.
